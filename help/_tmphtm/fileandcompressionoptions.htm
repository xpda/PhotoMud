File and Compression Options
 Previous  Next

  • JPG Compression -- the default compression used when saving .jpg files. This can be changed when you save a photo.
  • Email JPG Compression -- the default compression used when emailing .jpg files. This can be changed when you email photos.
  • TIFF compression mode -- This is the default compression mode to be used when you save TIFF files. They are all "lossless," which means that there is no loss of image quality when you save the file. Photo Mod can read most other TIFF compression formats.
  • Uncompressed --  Save TIFF files in uncompressed format. This makes a larger file, but it should be compatible with software that cannot read compressed TIFF files. You can also select the TIFF compression mode when you save the TIFF file.
  • LZW -- Save TIFF files using LZW compression. LZW is usually results in smaller files than packbits and uncompressed TIFF files, but it takes slightly longer to save.
  • Packbits -- Save TIFF files using packbits compression. LZW is usually results in larger files than with LZW compression but smaller than uncompressed TIFF files.
  • Save TIFF files in CMYK format -- CMYK is the color mode sometimes used in printing. When a photo is saved in CMYK and read back into Photo Mud, it is converted to RGB until it is saved again. You may get better final image quality if you save the photo in CMYK color after most of the editing is complete. CMYK can be selected when you save a TIFF file.
  • Use a transparency color with PNG files -- With a PNG file it is possible to name one color as the transparent color. This function of PNG files is not supported by some applications.
  • PNG transparent color -- Select the transparent color for a PNG file. Black is often used.

You can reset all Photo Mud options to the program defaults buy clicking the "Reset All Options" button at the bottom of the Options window.
