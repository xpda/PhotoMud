Selecting and Tagging Photos
 Previous  Next


You can select or tag more than one photo in Photo Mud Explorer to perform operations on multiple files. There are commands using multiple photos for printing photos, printing calendars, emailing photos, converting files, creating web pages, and displaying slide shows.

Thumbnail View

In Thumbnail view you can select multiple files with the mouse, and by holding down the shift or control key while you select additional photos.

A file is selected when it is highlighted. To select all the photos in the current folder, press Ctrl- A.

Details View

In Details view, instead of selecting multiple files, you "tag" them, or put a check mark in the check box to the left of the file name. In the Details view, you can tag photos from more than one folder (provided that option is enabled).

A file is selected when it is highlighted. A file is tagged when its check box (which appears only in Details view) is checked. You can tag all selected files, select all tagged files, or clear all tags by choosing the appropriate command under the Edit menu.

To select all the photos in the current folder, press Ctrl-A. To tag all the photos in the current folder, press Ctrl-A to select all photos then Ctrl-T to tag the selected photos.

After you perform an operation on a set of tagged photos, the tags may be cleared. The tags can also be cleared by pressing Esc. To restore a set of tags after they are cleared, use Set Previous Tags from the Edit Menu.
