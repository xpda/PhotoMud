Cropping the Image  
Previous  Next


Image Cropping is a two-step process. First, use Set Cropping Area under the Image menu or the Cropping Tool in the toolbar to specify the area of the image to be cropped.

To set the cropping area, drag the cropping box to the desired location, and drag the sides of the cropping box to set the size of the cropping area. The clear area inside the cropping box is the part of the photo that will be left after cropping is complete.

After the cropping box is set at the proper size and location, press Enter, click outside the cropping box, or select Crop under the Image menu to image to actually crop or trim the photo.
