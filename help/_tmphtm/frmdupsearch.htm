Search for Duplicate Photos
 Previous  Next


Photo Mud can search a folder for duplicate and similar photos. Photos of different types and sizes are matched based on the similarity of the images.

To search for duplicate and similar photos, select Search for Duplicates from the Tools menu. You can specify the folder to be searched, whether to search subfolders, and the tolerance of the selection process. Click the Search button to begin the search process. It may take some time, since each photo in the folder has to be loaded.

The searching process goes faster with a lower search tolerance.

When the search is complete, each pair of matched photos is displayed, along with the "difference score." A lower score means the photos are more nearly the same.

You can browse through the matched photos, and delete one of a pair of matches if you want to. The file name and photo size is displayed underneath each photo. You can click on a photo to display it in the full screen.

If there is more than one match for a photo, that photo will be listed with each of its matches.
