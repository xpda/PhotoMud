Selections
 Previous  Next


A selection is a section of an image that is to be changed. When a selection is active, most of the editing commands will operate only on that portion of the image. For example, you can select only the sky in a photo and make it brighter or darker without affecting the rest of the image.

The following selection commands are available under the Edit menu and in the toolbar:
  • Select Rectangle
  • Select Ellipse
  • Freehand Selection
  • Select Similar Colors
  • Invert Selection

A selection can be copied to the Windows Clipboard and pasted into the same photo or another photo.

When as selection is active, as indicated by the dashed line around the outside, you can set the feathering range. This makes the border of the selection fuzzy, so adjustments to the selection will fade into the rest of the photo. You can drag an active selection from one location to another.

After you are finished with a selection, you can click outside the selection or press Enter to save the selection into the drawing. Alternatively, you can press Esc to cancel changes to the selection.

The selected area in an image can be deleted, or replaced with the background color, using the Delete Selection command under the Edit menu of Photo Mud Editor.
