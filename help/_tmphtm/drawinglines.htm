Drawing Lines     
Previous  Next


You can draw lines, curves, circles, arrows, and rectangles using Photo Mud, by selecting those commands under the Draw menu.

You can also draw a freestyle line using the Sketch command under the Draw menu. To use the Sketch command, just hold the mouse button down while you draw on the photo.

The line, curve, and arrow commands are completed by double clicking or by pressing Enter.

You can specify the line style and thickness with Line Options under the Draw menu. The drawing color can be selected a number of ways.
