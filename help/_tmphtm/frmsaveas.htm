Saving Photos  
Previous  Next


Photo Mud can save a photo in any of several different file formats.

To save a photo, select the Save or Save As command under the File menu. With the Save As command you are able to specify options for .jpg, .tif, or .png files.

With Photo Mud Explorer you can optionally resize a photo when it is saved.

With Photo Mud Editor you can save a selected portion of a photo using Save Selection under the File menu.
