You can edit, save, or print a screen image with Photo Mud.

1. With the desired configuration on the screen, press Shift-PrtScr to copy the screen image to the Windows clipboard. The PrtScr key is to the right of the function keys on most keyboards.

2. In Photo Mud, press Ctrl-Q or select Paste into New Window from the Edit menu to copy the image in the clipboard into a new Photo Mud window.
