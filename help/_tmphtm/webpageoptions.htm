Web Page Options
 Previous  Next

  • Background color -- The background color of the web page.
  • Text color -- The color of text on the web page.
  • Number of thumbnails per row -- The number of thumbnail images in each row on the web page. The number of rows will be determined by this value and the number of photos selected or tagged.
  • Thumbnail size -- The width and height of thumbnail images, in pixels.
  • Drop shadow size -- A shadow can be added to the thumbnail images. A normal size is 6 pixels. Set this value to zero if you don't want a drop shadow.
  • Table border size -- The table border size is the thickness of the lines around the thumbnail images. Use zero if you don't want any lines. Zero, one or two are common values.
  • Space around cell contents -- This is the minimum number of pixels between the thumbnail image plus caption, and the edge of the cell in the table of thumbnails. It corresponds to the cellpadding html tag.
  • Space between cells -- This is the number of pixels between cells in the table of thumbnails. It corresponds to the cellspacing html tag.
  • Title text size -- The size of the title, in points. If you are not sure, you can check the Normal Size box.
  • Caption text size -- The size of the captions, in points. If you are not sure, you can check the Normal Size box.
  • Font name -- Enter the font name, such as Arial or Times New Roman.
  • Google Analytics code -- If you have a Google Analytics account, you can enter the account code to have the Java code added to the web page.
  • Convert photo times from UTC to local time -- If your camera was set to UTC (Universal Time Coordinated, or Greenwich Mean Time) when the photos were taken, you can check this box and Photo Mud will automatically adjust the times in the captions to the computer's local time. This only affects the captions, and not the photos themselves. It only affects the captions if you check the option in the Web Page command to add the time or date to the captions.

You can reset all Photo Mud options to the program defaults buy clicking the "Reset All Options" button at the bottom of the Options window.
