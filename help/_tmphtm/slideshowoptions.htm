Slide Show Options
 Previous  Next

  • Show slides in random order -- Display the photos in random order.
  • Order by file fame - Display the photos in file name order.
  • Show file name with slides -- Display the photo's file name at the bottom of the screen.
  • Show photo description with slides -- Display the photo description at the bottom of the screen.
  • Seconds per frame -- The number of seconds each photo is to be displayed.
  • Dissolve time -- The time for the dissolve transition between photos, in milliseconds. Use zero for no dissolve.

You can reset all Photo Mud options to the program defaults buy clicking the "Reset All Options" button at the bottom of the Options window.
