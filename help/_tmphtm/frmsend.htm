Emailing Photos  
Previous  Next


To email a photo select Email Photo, Email Selected Photos, or Email Tagged Photos from the File menu. The current photo(s) will be attached to a blank email. Add the recipient(s) and any text you want to the email, then send it as you would any email.

To email a number of images, select the images in Photo Mud Explorer and select Email Tagged Images from the File menu.

You can pick the option "Convert or Resize the Image" in the email dialog to change the image format or size in the email. This is convenient because many photos are rather large to be sent as email attachments.

With a .jpg image you can specify a higher compression to make the email image even smaller, although this lowers the image quality.
