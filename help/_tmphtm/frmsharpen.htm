Sharpening an Image
 Previous  Next


In many cases, an image that is a little blurry can be substantially improved with the Photo Mud sharpening function.

There are two sharpening functions available in Photo Mud, normal sharpening and the unsharp mask. The effectiveness of each depends on the photo being sharpened.

With normal sharpening, you can set the strength of the sharpening function, which is the strength of the adjustment, or the amount each modified pixel is changed.

When using the Unsharp Mask, these three parameters can be adjusted:
  • Amount -- The strength of the adjustment, or the amount each modified pixel is changed.
  • Radius -- The radius of a pixel's neighborhood. This tells how far the program will look in determining whether a pixel should be adjusted.
  • Threshold -- The threshold at which a pixel is modified; the minimum difference between a pixel and its blurred neighborhood.

The Sharpen command is under the Image menu of Photo Mud Editor.
