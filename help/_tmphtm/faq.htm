The Rename command under the Edit command of Photo Mud allows you to rename the selected file.

In Photo Mud Editor, you can press the F3 key to repeat the last command. For example, if you want to draw more than one line, you can draw one line using the Line command and then press F3 to draw subsequent lines.

Revert to Last Saved Copy, under the Edit menu of Photo Mud Editor, reloads the photo file and reverts back to the copy when you last saved the photo. This is useful if you have made a lot of changes to a photo, but then decide to start over.
