Fix a photo that is too red or too blue.
In Photo Mud Editor, select %Indk3onbkBrightness and Contrast%Indk3onbk from the Color menu. Then change the intensities for the specific colors.

Fix a photo that looks hazy or faded.
The best way is with usually to "Expand Midtones" with the %ID_6.z.ywHistogram Adjustment%ID_6.z.yw under the Color menu. You can also do more complex adjustments under the Histogram Adjustment. For a one-step enhancement, try the %IndcmvsjpAuto Enhance%Indcmvsjp command under the Color menu.

Trim or crop a photo.
Use %_45q9_Set Cropping Area%_45q9_ under the Image menu.

Use Photo Mud to display an image you double-click on in Windows Explorer or an email attachment.
Select %ID_a23x__Associate File Types%ID_a23x__ under the Tools menu, and check the file types you want to "associate" with Photo Mud.

Copy an object or person from one photo into another.
%Indbcgh98Select%Indbcgh98 the section of the picture to be copied using one of the selection methods under the Edit menu of Photo Mud Editor. Then press Ctrl-C to copy the selected area to the clipboard. Open the image where the selection will be copied, and use %a0eugPaste%a0eug or %a0eugStretch Paste%a0eug under the Edit menu to copy the selection into the new image.

Find out when a photo was taken.
Most modern digital photos contain information such as camera settings, date and time, and technical details of the image file. To display this information, select %con.vfv2_Image Information%con.vfv2_ from the Tools menu.

Add a caption or comments to a photo.
Select %con131t6uEnter File Comment%con131t6u from the Tools menu.
The comments can be displayed with %con.vfv2_Image Information%con.vfv2_, and can be automatically added to %brs22wweb pages%brs22w generated with Photo Mud.

Download images from a camera or a scanner.
Select %a711euFrom Scanner or Camera%a711eu from the File menu. If there is more than one scanner or camera attached to your computer, you can select the input device.

Rotate an image 90 degrees.
You can rotate images under Photo Mud Explorer using a single keystroke or mouse click for %Ind_rni6wRotate Left%Ind_rni6w (Ctrl-L) and %Ind_rni6wRotate Right%Ind_rni6w (Ctrl-R). These commands automatically save the rotated image.

Make a slideshow from a set of photos.
%5xpvplTag%5xpvpl the images you want to include in the slide show, then select %ID_h5_8f1Slideshow%ID_h5_8f1 in the Tools menu of Photo Mud Explorer.

Merge two photos.
Use the %4ol4mlCombine Images%4ol4ml command under the Tools menu of Photo Mud Editor, or by %a0eugcopying and pasting%a0eug from one image to another.

Email several photos at once.
%5xpvplTag%5xpvpl the images you want to email, then select %Ind6bk7nhEmail Tagged Images%Ind6bk7nh from the File menu.

Capture a screen image.
Select %KEY84nkq6Screen Capture%KEY84nkq6 from the File menu. Then bring up the screen you want to capture, press F11, and drag a box around the part of the screen you want to capture.
