Making a Web Page   
Previous  Next


Photo Mud can automatically create a web page of selected or tagged photos. The web page will contain a table of thumbnail images and optional captions. Any thumbnail image on the web page can be clicked on to view the full-size photo. Here's how to do it:
  1. In Photo Mud Explorer, tag or select the photos you want to be included in the web page.
  2. Select Make Thumbnail Web Page from the Tools menu.
  3. Enter any captions for the photos.
  4. Add any addition options for the web page.
  5. Click the "Save Web Page" button.
  6. Click the "View Web Page" button to see the result.

It's a good idea to save the web page in a separate folder. When you save a web page, three things are normally saved:
  • The HTML file. This is the web page itself. It contains references to the photos and the thumbnail photos.
  • The thumbnail photos. These are the small photos that appear in the web page, having the same name as the photo file, but with the prefix "small_".
  • The photo files. These are the full-sized photos, or they can be resized copies the originals.

After you have saved a web page, you can view it in your web browser by clicking on the "View Web Page" button. If you need to make changes, you may want to check the option "Save HTML file only, no photos," if the photos are correct.

Note:  You can make changes to the web page and save it again as long as you stay in the Web Page dialog. After you click "Close" and go back to Photo Mud Explorer screen you will no longer be able to made modifications to the web page using Photo Mud, although you can overwrite it or edit it with an html editor.

   


Thumbnail Photos

The thumbnail photos used in the web page are small copies of the original photos. They are copied to the same folder as the HTML file, with "small_" added onto the beginning of the original photo file name. When you make a web page, the thumbnail photos appear in the page. Then when you browse the web page, you can click on a thumbnail photo to see the large version of the photo.
Photos for the Web Page

In the lower lef corner of the Make Web Page screen, there is a set of boxes that represent the photos on the web page. You can click on any of the boxes to see the photo and optionally add a photo caption to be shown on the web page. You can also select a photo in the photo list above the photo arrangement boxes.

If you would like to reposition a photo, use the up and down arrow buttons to the right of the photo list. This will move the current photo up or down one position.

Photo Captions

You can enter a caption to be displayed under a photo on the web age underneath the photo's image. You can also select any of these items to be appended to the photo captions:
  • Photo Comments -- any photo comments in the photo file.
  • File Name -- the name of the photo file, not including the folder path.
  • Resolution -- the width and height of the photo, in pixels.
  • Photo Date -- the date the photo was taken, as stored in the photo file.
  • Photo Time -- the time the photo was taken, as stored in the photo file.
  • Hour Offset -- the number of house to add to the photo date and time. This is useful if your camera time was set to a different time zone than where the photos were taken. If the option has been selected to convert photo times from UTC to local time, the hour offset will be added to the photo time after the UTC-to-local conversion is done.
  • GPS Coordinates -- the GPS latitude and longitude stored in the photo file.This is added to the captions only for photo files containing GPS information.
  • GPS Altitude -- the GPS altitude stored in the photo file.This is added to the captions only for photo files containing GPS information.
  • Google Map Link -- a link to Google maps using the GPS latitude and longitude. This is added to the captions only for photo files containing GPS information.

Web Page Options

Title is the heading at the top of the page and the html web page title.

Several other web page options can be accessed by clicking on the Options button.
Save Web Page Photos

The small thumbnail photos that Photo Mud creates are always saved with the web page (unless you save html only). Then Photo Mud will ask if you want to save the web page photos. If so, the original photos will be copied to the folder with the web page itself. You can optionally resize the photos when they are copied.

The web page photos should normally be saved unless they are already in the folder where you save the web page.
