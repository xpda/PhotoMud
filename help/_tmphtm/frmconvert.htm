Converting Photos
 Previous  Next


Photo Mud can convert several images at once to a single file format, and can optionally resize the images in the process. This is useful if you have a number of images from a digital camera or scanner and you want to convert them to another format or reduce their size for email or a web page.

Here's how to do it:
  1. In Photo Mud Explorer, tag or select the photos to be processed.
  1. Select Convert Selected Photos or Convert Tagged Photos from the File menu.
  1. In the File Conversion dialog box, select the output file format. JPG is the most popular format for images.
  1. If you are saving the images as JPG files, you can select the JPG Compression.
  1. If you want to resize the images, enter the maximum height and width, in pixels. The images will maintain their aspect ratios. In other words, the images will not be stretched wider or taller; they will only become larger or smaller.
  1. Select the path for the images to be saved. The images will be saved under their current file names in the destination folder. If the destination folder is the same folder the original images are in, then they will be overwritten.
  1. Click the Start Conversion button.
  1. You can optionally specify a color adjustment to be applied to the images as they are converted. This makes it easy to modify a group of photos that are too dark, too red, etc. You can click the arrow buttons to browse through the tagged images and see the result of the color adjustment before you convert the files.

If Photo Mud Explorer is in the Thumbnails view, you select the images to be converted from the current folder. In the Details view, you can tag photos from more than one folder (provided that option is enabled). If you want these to be copied into individual folders instead of all into a single folder, you can check "Save Folders."

Technical Information you probably don't need to know

EXIF information is retained as images are converted, including most maker notes.
