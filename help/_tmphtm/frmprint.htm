Printing with Photo Mud  
Previous  Next


To Print with Photo Mud, use the Print command from the File menu. A number of standard printing options are available. You can print to any size and justification, print at landscape or portrait orientation, and specify values in inches or mm. A preview of the photo on paper is displayed. You can print multiple photos at once using Photo Mud Explorer.

The printer selection is at the top of the screen. You can click on the Preferences button to access printer settings such as quality and paper size.

If you select the option to "fit to page," the photo will be scaled as large as possible on the page. Alternatively, you can specify the size of the photo on the page in inches (or mm). You can print a single photo spanning several pages if you select the option "Print on Multiple Pages" and specify a size larger than the paper size.

The Actual Size button calculates the photo size on paper based on the Pixels per Inch value saved in the photo file.

If you are printing more than one copy or a group of photos, you can select the option to print multiple photos per page. You can then specify how many rows and columns of photos to be printed on each page, along with the border size. The border size is the empty space on the page around each photo.
