Supported File Types
 Previous  Next


Photo Mud can save photos and other images in any of these file formats:

JPEG                             .jpg, .jpeg, .jp2, .j2k, .jtf
TIFF                               .tif, .tiff
PNG                               .png
GIF                                .gif

FlashPix          .fpx
Photoshop            .psd
Targa                   .tga
Windows Bitmap        .bmp
Windows Cursor        .cur
Windows Icon            .ico

Some file formats have limitations in their size and color depth, so it may not be possible to save some photos in some of these file formats.

There are many variations on these file formats, and some formats are only partially supported.

You can select the file types to be displayed in Photo Mud Explorer with Select Image File Types under the View menu.
