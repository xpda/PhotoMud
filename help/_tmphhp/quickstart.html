Photo Mud Explorer and Editor
When you start Photo Mud, it loads the Photo Mud Explorer, which is handy for browsing through photos. To edit the photo on the screen, double click on it, press Enter, or click on the File Open tool on the left end of the toolbar. This loads the Photo Mud Editor. If you start Photo Mud by double clicking a photo file in Windows Explorer, it starts in Photo Mud Editor.

In Photo Mud Explorer you can view thumbnail images of photos, or a list of file names, sizes, and dates. You can drag the borders between sections in Photo Mud Explorer to adjust the look of your screen.

Full Screen Browsing
Use the full screen mode (Ctrl-M or the Full Screen tool) to view a photo on the entire screen. You can press Page Up or Page Down to move to the next or previous photo in the current folder.

Tagging photos
You can select or tag more than one photo in Photo Mud Explorer to perform operations on multiple files. This allows you to use multiple photos for printing photos, printing calendars, emailing photos, converting files, creating web pages, and displaying slide shows.

Photo Information
You can display detailed photo information from digital camera photos, and you can add internal comments to .jpg files.
