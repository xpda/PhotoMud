Photo Information
 Previous Top Next

The Photo Information function displays detailed information about photo. Most digital cameras include EXIF information in the photos that includes the camera model, date and time the picture was taken, shutter speed, etc.

The following information is shown for most photos:

·    Description-- The user comment is displayed if there is one. You can add a user comment to any .jpg, .tif, or .png photo.
·    File Type -- The file type includes more information than the file extension. For example, there are several types of .jpg files.
·    Image Size -- Height and width of the photo in pixels.
·    Output Resolution -- The X and Y resolutions are the number of pixels per inch horizontally and vertically. These are somewhat arbitrary values since the photo can be displayed and printed at any size.
·    Size on Disk -- The amount of space the photo occupies on disk.
·    Size in Memory -- The amount of RAM required to hold the photo while it is being used in a program.
·    Bits per Pixel -- The number of bits that make up each pixel in the photo.
·    Compression -- The type of compression used when the photo is saved on disk.

·    The following EXIF Information is commonly included for photos from modern digital cameras, although different cameras provide different information.

·    Date and Time -- Date and time the photo was taken.
·    Original Date and Time -- This is usually the same as the date and time the photo was taken, but some software modifies the first "Date and Time" field, leaving this one unchanged.
·    Date and Time Digitized -- This field is sometimes used when photos are scanned or digitized.
·    Make -- Camera manufacturer.
·    Model -- Camera model.
·    Software -- Internal camera software version.

·    Exposure Time -- Exposure time, in seconds
·    Exposure Bias -- The amount the exposure was bias above or below the normal exposure time.
·    Exposure Program
·    Exposure Mode -- Auto, manual, etc.
·    F-Number
·    Max Aperture
·    ISO Speed

·    Focal Length - Actual focal length
·    Focal Length (35mm) -- Focal length equivalent to a 35mm camera
·    Digital Zoom -- Whether digital zoom was used.
·    Contrast -- Camera's contrast adjustment
·    Saturation -- Camera's saturation adjustment
·    Sharpness -- Camera's sharpness adjustment
·    White Balance -- White balance setting
·    Light Source
·    Scene Capture Mode -- Automatic scene mode setting, such as sun, clouds, beach, etc.
·    Metering Mode -- Light metering mode, such as point, weighted average, etc.
·    Flash -- Whether and how the flash was fired.

· Maker Note -- The Maker Note area of the photo may include additional camera parameters and settings. Maker notes are proprietary manufacturer information. The data is not well documented, and the information may vary among camera models. In other words, information presented in the maker notes may be unreliable.

On the Photo Information screen you can browse from one photo to the next in the current folder by clicking on the arrow buttons beneath the file name.

If a digital photo is saved by a program that does not support EXIF information, that information may be lost when it is saved. The EXIF information is available in .jpg and .tif files.

You can click the Copy button to copy this information to the clipboard so you can paste it into other applications, and you can click the Save button to save the information into a text file.

Photo Information is under the Tools menu.
