Adding a Border to an Image
 Previous Top Next

You can add a 2D or 3D border to an photo using Border under the Image menu . The border is actually a colored extension of the photo on for sides, so it will appear with other applications.

You can add a single line around the outside of the photo with the Single-Line option.

The Buttonize option makes a small 2-pixel border around the image similar to Windows buttons.

Border is in the Image menu of Photo Mud Editor.
