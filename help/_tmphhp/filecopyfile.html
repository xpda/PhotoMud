Copying a File on Disk
 Previous Top Next

In Photo Mud Explorer you can copy an image file from one location to another on the hard drive by selecting Copy Image File under the File menu. This is similar to loading the file into memory and then saving it to a new location, but .jpg files are not modified by this operation like they are when they are saved.

Copy Image File is under the File menu and should not be confused with the clipboard copy functions under the Edit menu.
