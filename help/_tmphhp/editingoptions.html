Editing Options
 Previous Top Next

·      Enable multi-level undo -- Multi-level undo allows you to undo and redo several operations. However, this can slow down the program if you are working with very large photos.

·      Begin Editing at 1:1 Zoom -- If this is checked, then an image loaded into Photo Mud Editor will be displayed at 1:1 zoom, where each pixel on the display shows one pixel of the image. If this is not checked, the image will be displayed at the largest zoom so that it fits on the screen.

·      Tolerance for Similar Color Selection -- The Color Tolerance is the range of pixel intensities to be used with Fill and Select Similar Color. The larger the number, the wider the selection in the image. There is an upper and a lower tolerance. The upper tolerance specifies the range of colors to be selected that are brighter than the color at the designated point. The lower tolerance specifies the range of colors to be selected that are darker than the color at the designated point.

·      Save keywords in photo comments field -- Leave this unchecked to use the Windows keywords field for Photo Comments, check to add keywords to the Exif Comment field. This only applies if the Windows keywords field is enabled.

·      Include Windows comment fields -- Select to enable the Windows image comment fields Keywords, Comments, Title, Subject, and Author in Photo Comments.

·  Edit comment keywords -- Add or remove keywords from the list in Photo Comments.

You can reset all Photo Mud options to the program defaults buy clicking the "Reset All Options" button at the bottom of the Options window.
