Customizing the Toolbar
 Previous Top Next

The toolbar is the row of buttons below the menu at the top of the screen. You can specify the size of the buttons (or icons) and also which buttons should be displayed in the toolbar.

Photo Mud Explorer toolbar:


To customize your toolbar, select Customize Toolbar from the Tools menu or from the menu that appears when you right-click on the toolbar.

The commands in the toolbar are listed on the right, and the commands available for the toolbar are listed on the left. You can move a command from one to the other by selecting the commands to move and using the Add or Remove command buttons between the lists of commands.

A separator is a small space you can place between buttons. Each command can be used only once in the toolbar, but you can use more than one separator.

You can move the buttons around on the toolbar using the Move Up and Move Down buttons to the right of the toolbar list.

You can choose small or large icons at the bottom of the dialog box.

You can also select the option to "show toolbar text." This will cause the name of each command in the toolbar to appear underneath its button.

Toolbar with Text Captions:

When the captions are displayed with the toolbar, there may not be enough room on the screen to show all the buttons. When this happens, you can access the rest of the toolbar using a small drop-down menu at the right end of the tool bar.

If you want to go back to the original toolbar configuration, press the Reset button. This goes back to the original program defaults.
