Merging Color Separations
 Previous Top Next

Photo Mud can merge separate black and white images that represent different colors into a single color image. To do this, first select or tag the files you want to merge. These should be black and white images of the same image, each representing a different color or color bandwidth. For example, you might have three images, one for red, one for green, and one for blue.

After the images are selected, click on Merge Color Separations from the Tools menu to bring up the Color Merge dialog. Each of the tagged file names is listed. Next to the file name, you can specify the RGB values to be used with that file. You can click on the color names or you can enter the values for Red, Green, and Blue. The resulting color image is displayed as you enter the colors.

The overall brightness or intensity of the resulting color image is automatically adjusted if you check the "Normalize Weights" option.

You can merge up to 10 images with this command.

NASA and ESA spacecraft often return a number of black and white images of the same object, filtered at various light wavelengths. This command is useful in converting a set of those images into a single color image.

For reference, here are some sample color wavelengths:

It is also possible to combine images from non-visible areas of the electromagnetic spectrum.
