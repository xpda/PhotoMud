Slide Show
 Previous Top Next

To make a slide show of various photos select or tag the photos you want to display and then select Slide Show under the Tools menu.

The slide show controls will appear at the bottom of the screen when the slide show starts and whenever the mouse is moved.

The control panel has buttons for Back, Pause/Play, Forward, and Stop. The number of seconds each photo is displayed can be specified on the control panel.

You can click on the Options button to configure the these slide show options:

·     Order: random, file name, file date, or photo date
·     Show file name with slides
·     Show photo description with slides
·     Show photo date with slides
·     Frames per second
·     Dissolve time
