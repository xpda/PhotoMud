Photo Mud Editor Menus

File Menu
New -- Create a new image in Photo Mud Editor.
Open -- Open a file in Photo Mud Editor.
Close -- Close the current photo.
Close All -- Close all open photos in Photo Mud Editor.
Save -- Save the currently open photo file.
Save As -- Save the current photo to a specified file.
Save Selection As -- Save the selected part of the current photo to a specified file.
Scan -- Scan a photo or document and load it into Photo Mud Editor.
Print -- Print the current photo.
Email Photo -- Email the current photo.
Explore Folders -- Open Photo Mud Explorer.
Exit - Exit the program.

Edit Menu
Undo -- Undo the last operation.
Redo -- Redo the previously undone operation, or Undo the last Undo.
Revert to Last Saved Copy -- Reload the last saved copy of the photo.
Repeat Last Operation -- Repeat the last operation.
Copy -- Copy the current photo or selection to the Windows Clipboard.
Cut -- Cut (copy, then delete) the current selection to the Windows Clipboard.
Paste -- Paste the image in the clipboard into the current photo.
Paste into New Window -- Paste the image in the clipboard into a new Photo Mud Editor window.
Stretch Paste -- Paste the image using four corner points.
Delete Selection -- Delete the current selection.
Delete File -- Delete the current photo from disk.
Select Rectangle -- Select a rectangular area in the photo.
Select Ellipse -- Select an elliptical area in the photo.
Freehand Selection -- Select an area inside a line you draw.
Select Similar Colors -- Select an area in the photo that has similar colors to a designated point.
Invert Selection -- Select the area in the photo that is not currently selected, and deselect the rest.
Insert Page -- Add a page to the current image.
Delete Page -- Remove a page from the current image.

Image Menu
Resize -- Change the size of the photo.
Expand -- Expand the area of a photo by added blank space.
Rotate -- Rotate the photo.
Flip Horizontally -- Flip the photo horizontally to make a mirror image.
Flip Vertically -- Flip the photo vertically to make a mirror image.
Stretch -- Stretch a photo by picking four points for its corners.
Blur -- Blur a photo.
Sharpen -- Sharpen a photo.
Median Filter Noise Removal -- Remove noise using a Median Filter.
Edge Detection -- Detect edges in the photo using a selection of methods.
Red Eye Removal -- Remove red eyes resulting from a flash photo.
Special Effects -- Apply selected special effects to a photo.
Artistic Effects -- Apply selected artistic effects to a photo.
Border -- Add a border to a photo.
Set Cropping Area -- Select an area in the photo to be cropped.
Crop -- Finalize the cropping operation.

Color Menu
Auto Enhance -- Automatically perform a color and contrast correction.
Brightness, Contrast, Saturation -- Adjust the Brightness, Contrast, and Saturation of a photo.
Color Adjustment -- Adjust the red, green, and blue values for shadow, midtones, and highlights.
Histogram Adjustment -- Perform a variety of color and intensity adjustments on a photo.
Hue, Saturation, Intensity -- Adjust the Hue, Saturation, and Intensity of a photo.
Light Amplifier -- Multiplies the intensity of an photo.
Halftone -- Converts a photo to a halftone image.
Grayscale -- Converts a photo from color to grayscale.
Negative -- Makes a negative image of a photo.
Sepia -- Converts a photo to sepia, making it look antique.

Draw Menu
Text -- Add text to the current photo.
Line -- Draw a line on the current photo.
Curve -- Draw a curve on the current photo.
Arrow -- Draw an arrow on the current photo.
Circle -- Draw a circle on the current photo.
Box -- Draw a rectangle on the current photo.
Line Options -- Select the line pattern (dashed, dotted, etc.) and thickness.
Sketch -- Draw a freehand line on the current photo.
Fill Area of Similar Color -- Fill an area in the photo that has similar colors to a designated point.
Fill Selected Area -- Fill an the selected area in the photo with a solid color.
Drawing Color -- Select the drawing (foreground) color.
Background Color -- Select the background color.
Color Sample -- Get the color of a point in the photo.

Tools Menu
Photo Information -- Display information about a photo, including comments and camera settings.
Enter Photo Comments -- Enter photo comments to be saved with the file.
Search for Duplicates -- Search for duplicate photos on the hard drive.
Combine Photos -- Combine two photos.
Photo Mosaic -- Create a photo mosaic made up of many small individual photos.
Filter Workshop -- Apply custom image filters to a photo.
Associate File Types -- Select file types to be associated with Photo Mud.
Set as Wallpaper -- Set the current photo as Windows Wallpaper.
Customize Toolbar -- Select tools for the toolbar, and other toolbar options.
Options -- Set Photo Mud options and preferences.

View Menu
Zoom In -- Zoom in on the current photo.
Zoom Out -- Zoom out on the current photo.
Zoom Window -- Zoom into a rectangle dragged on the photo.
Full Screen -- View the current photo on the full screen.
Show Toolbar -- Show or hide the toolbar.
Refresh -- Refresh the screen.
Fit to Window -- Zoom the photo so it fits in the window at its maximum size.
25% Zoom -- Show the photo at 25% zoom (1/4 size).
50% Zoom -- Show the photo at 50% zoom (1/2 size).
100% Zoom -- Show the photo at 100% zoom (1 photo pixel = 1 screen pixel).
200% Zoom -- Show the photo at 200% zoom. (1 photo pixel = 2 screen pixels).

Window Menu
The window menu allows you to select any open window -- any open photo or Photo Mud Explorer.

Help Menu
Help -- Show the Help Contents.
Help Index -- Show the Help Index.
Show Photo Mud Tips -- Show the Photo Mud Tips.
Register Photo Mud Online -- Go to the Photo Mud web site and register the program.
Check for Updates Online -- Go to the Photo Mud web site and check for updates.


Photo Mud Explorer Menus

File Menu
Open -- Open a file in Photo Mud Editor.
Edit Selected File(s) -- Edit the selected file(s) in Photo Mud Editor.
Save As -- Save the selected photo to a specified file.
New -- Create a new image in Photo Mud Editor.
Scan -- Scan a photo or document and load it into Photo Mud Editor.
Print -- Print the selected or tagged photo(s).
Email Photo(s) -- Email the selected or tagged photo(s).
Copy File to Disk -- Copy the selected file to another location on the hard drive.
Convert Photos -- Convert photos from one format to another, optionally resizing them.
Photo Mud Editor -- Show Photo Mud Editor.
Exit - Exit the program.

Edit Menu
Copy File to Clipboard -- Copy the selected file to the Windows Clipboard.
Copy Image to Clipboard -- Copy the selected file's image or picture to the Windows Clipboard.
Copy File Name to Clipboard -- Copy the selected file's name to the Window Clipboard.
Paste File -- Paste the file in the clipboard to the currently selected folder.
Paste Image into New Window -- Paste the image in the clipboard into a new Photo Mud Editor window.
Delete File -- Delete the currently selected file(s) from the hard drive.
Rename File-- Rename the selected file.
Rotate Left -- Rotate the selected photo 90° to the left.
Rotate Right -- Rotate the selected photo 90° to the right.
Tag Selected Files -- Tag all selected files in the current folder.
Select Tagged Files -- Select all tagged files in the current folder.
Clear All Tags -- Clear all tagged files.
Set Previous Tags -- Tag the files that were tagged before the previous operation.
Select All -- Select all files in the current folder.

View Menu
Up One Level -- Move up to the next higher folder, the parent folder of the current folder.
Previous Folder -- Move to the previous folder.
Next Folder -- Move to the next folder (after a move to the previous folder).
Full Screen -- View the selected photo on the full screen.
Show Toolbar -- Show or hide the toolbar.
Refresh -- Refresh the screen.
View Thumbnails -- Select the Thumbnails view style.
View Details -- Select the Details view style.

Tools Menu
Photo Information -- Display information about a photo, including comments and camera settings.
Enter File Comments -- Enter photo comments to be saved with the file.
Make Thumbnail Web Page -- Make a web page out of selected or tagged photos.
Set as Wallpaper -- Set the current photo as Windows Wallpaper.
Slide Show -- Show a slide show of selected or tagged photos.
Merge Color Separations -- Merge color separations of a photo.
Photo Search -- Search Photo comments on the hard drive.
Search for Duplicates -- Search for duplicate photos on the hard drive.
Associate File Types -- Select file types to be associated with Photo Mud.
Select Photo File Types -- Select file types to be displayed in Photo Mud Explorer.
Print Calendar -- Print a monthly calendar using selected photos.
Customize Toolbar -- Select tools for the toolbar, and other toolbar options.
Options -- Set Photo Mud options and preferences.

Window Menu
The window menu allows you to select any open window -- any open photo or Photo Mud Explorer.

Help Menu
Help -- Show the Help Contents.
Help Index -- Show the Help Index.
Show Photo Mud Tips -- Show the Photo Mud Tips.
Register Photo Mud Online -- Go to the Photo Mud web site and register the program.
Check for Updates Online -- Go to the Photo Mud web site and check for updates.
