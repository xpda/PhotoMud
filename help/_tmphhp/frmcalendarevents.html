Custom Events for Calendars
 Previous Top Next

Custom events can be entered for your calendar by clicking on the Edit button at the top of the list of calendar categories.

To add an event, click on the first blank description and date listed, and add the description and date. For example, to add a birthday, you might put "John Smith's birthday" in the description and "10/22/1970" for the date.

The text you enter in the description will be printed on that date in the calendar.

There are two fields you can use in the description to display the number of years:

<years> will display the number of years since the date. For example, you could enter a description of "Mary and Joe have been married <years> years today" with a date of 4/25/1998. On April 25 in the 2010 calendar, it will print "Mary and Joe have been married 12 years today."

<yth> is similar to <years> except for the formatting. <yth> is used for the nth year, based on the date. For example, you might use "Today is Dad's <yth> birthday" with a date of 2/28/1968. Then, on February 28 in the 2010 calendar, it will print "Today is Dad's 42nd birthday."

Custom events are automatically saved when the program is exited.
