Multipage Images
 Previous Top Next

Some image file types, such as .gif, .tiff, and .pcx, are capable of handling more image per file. These are called multipage images. Multipage .gif files are commonly used for web animations. Multipage .tiff files sometimes contain multiple resolutions of the same image.

If you open a multipage file, the multipage panel will be displayed at the lower right of the screen in Photo Mud Editor.

  

You can Insert and Delete pages, and you can show the pages in an animation at 10 pages per second.

You can also use Insert Page and Delete Page from the Edit menu to add and delete pages from an image.

If you save a multipage image to a file such as .jpg that does not support multiple pages, only the currently displayed page will be saved.

Photo Mud editing and drawing operations are performed only on the currently displayed page of a multipage file.
