Photo Comments
 Previous Top Next

You can save your own comments inside any .jpg, .tif, or .png file by selecting Enter File Comment under the Tools menu.

The Photo Description field is the most common field for photo comments, and can be automatically added to the photo caption in a thumbnail web page.

The Windows comment fields Keywords, Comments, Title, Subject, and Author can be enabled for .jpg files under the Edit options. These fields are not standard and are not supported in many applications, but they have the advantage of being used in Windows searches.

There is a list of keywords with check boxes on the Photo Comments screen to make it easy to add commonly used keywords to photos. To edit the keyword list, click the button to "Edit Keyword List" and enter up to 30 keywords you want to use, each on a separate line.

To add keywords to a photo, check the keywords to be added and click the "Add Checked Keywords" button. The keywords will be then added to the photo and will be available for searches. Keywords can also be entered and edited manually. The keywords are added to the end of the Photo Description unless the Windows fields are enabled, in which case they are added to the Keywords field.

You can browse through the photos in a folder using the arrow keys at the lower left of the File Comments screen. This makes it easy to add comments to several pictures at once. When you click on an arrow key to move from one photo to the next, the comments of the current photo are saved and the next photo is displayed.

The F3 key can be used to duplicate comments from the previous photo, and the F2 key can be used to clear the comment fields of a photo.

To display comments and other photo information, use Photo Information under the Tools menu.
