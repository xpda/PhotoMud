JPG Compression
 Previous Top Next

The JPG file format uses "lossy" compression so the image loses a little accuracy when it is compressed. This is loss of image accuracy is not normally noticeable at compression levels of 15 or below unless the file is saved and reloaded repeatedly. The compression value can range from 2 to 255.

Here are some sample JPG compressed images, at 2x zoom (each photo pixel occupies 4 screen pixels). As you can see, there is no noticeable difference in the original and the copy saved with a compression value of 15, but 100, 200, and 255 exhibit progressively worse quality.
