Resizing a Photo
 Previous Top Next

A photo can be resized using Resize under the Image menu of Photo Mud Editor. You can specify the number of horizontal and vertical size in pixels, or enter the percentage of the existing size for the new size -- 200% for twice as large, 100% for the same size, etc.

If "Maintain Aspect Ratio" is checked, then the resulting image will have the same aspect ratio, or ratio of width to height, as the original photo. If this is unchecked, then the resulting image may be a different shape than the original.

In addition to the photo size, the horizontal and vertical dots per inch can be specified. This will not affect the size of the photo, but it is used by some printing and desktop publishing applications to size a photo on paper. The dots per inch can optionally be used by Photo Mud to determine the size for printing.

You can resize more than one photo at once using Convert Selected Photos or Convert Tagged Photos from the File menu of Photo Mud Explorer.
