Photo Mud Explorer Options
 Previous Top Next

·     File listing view style -- Explorer can display "Thumbnails," small copies of the photos in a folder, or it can display detailed file information, "Details." Several functions in Photo Mud Explorer operate on several photos. In Details view you can tag the photos with a checkbox and select photos from more than one folder at a time. In Thumbnails view you can select only photos in the current folder.

·     Thumbnail width and height -- This is the size of the images displayed in Thumbnails view. Many digital cameras produce a thumbnail image that is 160x120. If you specify a thumbnail size 160x120 or smaller, the thumbnail images may be displayed significantly faster because the program will not have to generate new thumbnail images on the fly.

·  Allow tags in multiple folders -- This option enables photos to be tagged in more than one folder at a time. This only applies when the viewing files in Details mode. In Thumbnails mode, files can be selected only from a single folder at a time.

You can reset all Photo Mud options to the program defaults buy clicking the "Reset All Options" button at the bottom of the Options window.
