Rotating a Photo     
Previous Top Next

With Photo Mud Explorer you can rotate a photo 90 degrees left or right using the toolbar or Rotate Left (Ctrl-L) and Rotate Right (Ctrl-R) under the Edit menu. These automatically save the rotated photo. Most .jpg files are rotated and saved using lossless transformations, that is, without losing any image quality.

With Photo Mud Editor you can rotate an image any angle, using Rotate under the Image menu.

You can use the slider control or enter the angle to rotate. When entering an arbitrary angle, a positive value is rotated clockwise and negative goes counterclockwise.

Grid lines can be displayed during rotation to make it easy to align vertical or horizontal lines in the photo, such as the horizon or building walls.

When you rotate an photo at an arbitrary angle, Photo Mud can optionally trim the result so that none of the blank background shows in the corners. Select the "Trim Photo" option to do this.

If you prefer to have the entire photo appear in the rotated result, you can leave the "Trim Photo" option unchecked. If you want the blank area of the corners to be a specific color, set the background color before rotating the photo.
