Copy and Paste with the Windows Clipboard
 Previous Top Next
Copy  

In the Photo Mud Editor you can copy a photo or part of an photo to the Windows Clipboard using Ctrl-C or Copy under the Edit menu. If the current photo has an active selection, only the selected part of the photo will be copied to the Windows Clipboard.

After a photo is copied to the Windows Clipboard, it can be pasted into another Photo Mud image or into another application.

In Photo Mud Explorer you can copy a file, a filename, or the photo itself by selecting the appropriate copy function under the Edit menu or under the context menu that appears when you right-click on a file.

Paste  

To paste an photo or other image using Photo Mud Editor, use Ctrl-V or Paste under the Edit menu. Any image in the Windows Clipboard will then be pasted into the current Photo Mud photo. The pasted image can come from Photo Mud or from another application.

Stretch Paste under the Edit menu can be used if you want to scale or stretch the image in the clipboard into the current image.

When an image is pasted into Photo Mud, it comes in as a selection. You can drag it into position, and then press Enter to complete the Paste operation. If you get it wrong, remember that you can use Ctrl-Z for Undo.

In Photo Mud Explorer you can paste an image file into the current folder using Paste File under the Edit menu.

You can paste a photo or image from the clipboard into a new Photo Mud Editor window using Paste Image into New Window under the Edit menu. This is useful if you want to capture the screen image using Shift-PrtScr and edit, save, or print it.
