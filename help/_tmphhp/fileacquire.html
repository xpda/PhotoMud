Photo Mud supports most scanners. To load an image from a scanner, select Scan from the File menu. If you have more than one scanner attached to your computer, you will be able to select the device you want to use.

Technical Information you probably don't need to know
Photo Mud supports the Twain interface and the Windows WIA interface for image acquisition. If a scanner supports both these interfaces, you may see two choices for the same device, each with a slightly different user interface.

You can use this function to download photos from many digital cameras. However, when downloading a number of pictures from a digital camera it is usually faster to copy directly from the flash card to your hard drive using Windows Explorer.
