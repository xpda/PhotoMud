Resizing the Image

An image can be resized by specifying the number of horizontal and vertical size in pixels, or by entering the percentage of the existing size for the new size -- 200% for twice as large, 100% for the same size.

If "Maintain Aspect Ratio" is checked, then the resulting image will have the same aspect ratio, or ratio of width to height, as the old image. If this is unchecked, then the resulting image may be a different shape than the original.

You can resize a group of tagged images using the File Convert command in Photo Mud Explorer.

The Resize command is under the Image menu of Photo Mud Editor.
