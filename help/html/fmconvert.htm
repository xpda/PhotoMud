Converting Tagged Images

Photo Mud can convert several images at once to a single file format, and can optionally resize the images in the process. This is useful if you have a number of images from a digital camera or scanner and you want to convert them to another format or reduce their size for email or a web page.

Here's how to do it:

  • In Photo Mud Explorer, Tag the images to be processed.

  • Select Convert Tagged Images from the File menu.

  • In the File Conversion dialog box, select the output file format. JPG is the most popular format for images.

  • If you are saving the images as JPG files, select the JPG Compression.

  • If you want to resize the images, enter the maximum height and width, in pixels. The images will maintain their aspect ratios. In other words, the images will not be stretched wider or taller; they will only become larger or smaller.

  • Select the path for the images to be saved. The images will be saved under their current file names in the destination folder. If the destination folder is the same folder the original images are in, then they will be overwritten.

  • Click the Start Conversion button.

  • You can optionally specify a color adjustment to be applied to the images as they are converted. This makes it easy to modify a group of photos that are too dark, too red, etc. You can click the arrow buttons to browse through the tagged images and see the result of the color adjustment before you

The files will be copied to a single folder, even if the tagged files are in more than one folder.

Technical Information you probably don't need to know

EXIF information is retained as images are converted, including most maker notes.
