Photo Mud Quick Start

Photo Mud Explorer and Editor

When you start Photo Mud, it loads the Photo Mud Explorer, which is handy for browsing through images. To edit the image on the screen, press Enter or click on the File Open tool on the left end of the toolbar. This loads the Photo Mud Editor. If you start Photo Mud by double clicking an image file in Windows Explorer, it starts in Photo Mud Editor.

In Photo Mud Explorer you can view image thumbnails, detailed information, or just the image names. You can also drag the borders between sections to adjust the look of your screen.

Full Screen Browsing

Use the full screen mode (Ctrl-M or the Full Screen tool) to view an image on the entire screen. You can press Page Up or Page Down to move to the next or previous image in the current folder.

Tagging Images

You can use use tags, or check marks, next to the image in Photo Mud Explorer to perform operations on multiple files. There are commands for printing, emailing, converting, viewing, and creating web pages from tagged images.

Image Information

You can display detailed image information from digital camera images, and you can add internal comments to .jpg files.
