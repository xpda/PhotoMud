Importing an Image from a Scanner or Camera

Photo Mud supports most scanners and cameras for loading images. To load an image from a scanner or camera, select Import from Camera or Scanner from the File menu. If you have more than one scanner attached to your computer, you will be able to select the device you want to use.

Technical Information you probably don't need to know

Photo Mud supports the Twain interface and the Windows WIA interface for image acquisition. If a scanner supports both these interfaces, you may see two choices for the same device, each with a slightly different user interface.

When downloading a number of pictures from a digital camera, it might be faster to copy directly from the flash card to your hard drive using a flash adapter and Windows Explorer.
