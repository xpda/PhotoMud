Zooming In and Out

You can zoom an image in and out in Photo Mud Editor using the + and - keys. The = key also works in place of the + key. The Zoom commands under the View menu and the toolbars can also be used.

The Zoom Window command lets you drag a box around the part of the image you want to view. Fit to Window sizes the image so that it fills the editing window.

In Photo Mud Editor and Explorer, you can Press Ctrl-M to view the image in full screen mode, or use the Full Screen command under the View menu. Press any key to return to the normal mode. While in the full screen mode, you can press Page Up or Page Down to browse throught the images in the current folder.
