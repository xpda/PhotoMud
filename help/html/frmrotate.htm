Rotating the Image

With Photo Mud Editor you can rotate an image any angle. When entering an arbitrary angle, a positive value is rotated clockwise and negative goes counterclockwise.

The Rotate command is under the Image menu.

You can also rotate images under Photo Mud Explorer using the Rotate Left (Ctrl-L) and Rotate Right (Ctrl-R) under the Edit menu. These commands automatically save the rotated image. Most .jpg images are saved using lossless transformations.
