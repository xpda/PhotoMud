How To...

Fix a photo that's too dark or too light.

In Photo Mud Editor, select Brightness and Contrast from the Color menu.

Make a picture less blurry.

Use the Sharpen command under the Image menu of Photo Mud Editor.

Print several pictures on one page.

First, tag the images you want to print in Photo Mud Explorer. Then, under the Print command, check "Multiple Images per Page" and specify the number of rows and columns you want to print.

Make a web page of photos.

In Photo Mud Explorer, tag the images you want to include in the web page. The select Make Thumbnail Web Page from the Tools menu.

Fix a photo that is too red or too blue.

In Photo Mud Editor, select Brightness and Contrast from the Color menu. Then change the intensities for the specific colors.

Fix a photo that looks hazy or faded.

The best way is with usually to "Expand Midtones" with the Histogram Adjustment under the Color menu. You can also do more complex adjustments under the Histogram Adjustment. For a one-step enhancement, try the Auto Enhance command under the Color menu.

Trim or crop a photo.

Use Set Cropping Area under the Image menu.

Use Photo Mud to display an image you double-click on in Windows Explorer or an email attachment.

Select Associate File Types under the Tools menu, and check the file types you want to "associate" with Photo Mud.

Copy an object or person from one photo into another.

Select the section of the picture to be copied using one of the selection methods under the Edit menu of Photo Mud Editor. Then press Ctrl-C to copy the selected area to the clipboard. Open the image where the selection will be copied, and use Paste or Stretch Paste under the Edit menu to copy the selection into the new image.

Find out when a photo was taken.

Most modern digital photos contain information such as camera settings, date and time, and technical details of the image file. To display this information, select Image Information from the Tools menu.

Add a caption or comments to a photo.

Select Enter File Comment from the Tools menu.

The comments can be displayed with Image Information, and can be automatically added to web pages generated with Photo Mud.

Download images from a camera or a scanner.

Select From Scanner or Camera from the File menu. If there is more than one scanner or camera attached to your computer, you can select the input device.

Rotate an image 90 degrees.

You can rotate images under Photo Mud Explorer using a single keystroke or mouse click for Rotate Left (Ctrl-L) and Rotate Right (Ctrl-R). These commands automatically save the rotated image.

Make a slideshow from a set of photos.

Tag the images you want to include in the slide show, then select Slideshow in the Tools menu of Photo Mud Explorer.

Merge two photos.

Use the Combine Images command under the Tools menu of Photo Mud Editor, or by copying and pasting from one image to another.

Email several photos at once.

Tag the images you want to email, then select Email Tagged Images from the File menu.

Capture a screen image.

Select Screen Capture from the File menu. Then bring up the screen you want to capture, press F11, and drag a box around the part of the screen you want to capture.
