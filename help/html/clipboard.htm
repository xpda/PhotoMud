Copying and Pasting with the Windows Clipboard

Copying

You can copy an image or part of an image to the Windows Clipboard using Ctrl-C or Copy under the Edit menu. If the current image has an active selection (in Photo Mud Editor), only the selected part of the image will be copied to the Windows Clipboard.

After an image is copied to the Windows Clipboard, it can be pasted into another Photo Mud image or into another application.

In Photo Mud Explorer you can copy a file, a filename, or the image itself by selecting the appropriate command under the Edit menu or under the context menu that appears when you right-click on a file.

Pasting

To paste an image into Photo Mud, use Ctrl-V or Paste under the Edit menu. Any image in the Windows Clipboard will then be pasted into the current Photo Mud image. The image can come from Photo Mud or from another application.

Stretch Paste under the Edit menu can be used if you want to scale or stretch the image in the clipboard into the current image.

When an image is pasted into Photo Mud, it comes in as a selection. You can drag it into position, and then press Enter to complete the Paste operation. If you get it wrong, remember that you can use Ctrl-Z for Undo.

Pasting is done under Photo Mud Editor and not Photo Mud Explorer.
