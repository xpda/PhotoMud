What's New in Version 4.0?

There are hundreds of improvements in Photo Mud version 3.0. Here are a few:

  • Photo Mud Explorer

  • Flexible image browsing with Photo Mud Explorer

  • Automatic web page creation, with thumbnails and multiple images

  • Print multiple images with one or more images per page

  • Email multiple images

  • Convert, resize, or color-adjust multiple images in a single operation.

  • Flexible image tagging for multi-file operations

  • Slideshow

  • Search for duplicate and similar images

  • Faster program speed in many areas

  • Improved user interface in several commands

  • Improved Screen Capture

  • Images can be stretched to fit inside any four arbitrary points

  • Transparent backgrounds for selection copy and paste

  • Improved Sharpen command

  • Median filter noise removal

  • Improved functionality in edge detection

  • Red eye removal

  • Simplified brightness and contrast adjustment

  • Histogram advanced color adjustment

  • Auto-Enhance color adjustment

  • Light amplifier

  • Improved text

  • Improved 3D shape drawing

  • Improved image merging

  • Display internal Exif image information

  • Add Exif and Windows file comments to images

  • Customizable toolbars

  • Improved special effects:
    • Emboss

    • Posterize

    • Mosaic

    • Solarize

    • Intensity Detect

    • Random Noise
  • Revert to Last Saved command

  • Image browsing in full screen mode.

  • Drag-and-drop capability

  • Mozilla compatibility

  • Zoom Window command

  • Tips of the Day

  • Search for Photos based on Comments and Keywords
