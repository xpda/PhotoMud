Using the Toolbar

The toolbar just underneath the menu can be used to execute commands such as Save, Print, and Email. To see which command an icon represents, move the mouse to the icon for a moment and a tool tip will appear with the command name.

You can set the icon size and customize the tools that appear in the toolbar by right-clicking on the toolbar, or by selecting Customize Toolbar under the Tools menu.

Photo Mud Explorer and Photo Mud Editor use separate toolbars, and either toolbar can be customized.

You can remove the toolbar from the window to make more room for the image if you want to. The toolbar can be enabled or disabled under the View menu.
