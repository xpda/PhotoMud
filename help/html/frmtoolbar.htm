Customizing the Toolbar

The toolbar is the row of buttons below the menu at the top of the screen. You can specify the size of the buttons (or icons) and also which buttons should be displayed in the toolbar.

To customize your toolbar, select Customize Toolbar from the Tools menu or from the menu that appears when you right-click on the toolbar.

The commands in the toolbar are listed on the right, and the commands available for the toolbar are listed on the left. You can move a command from one to the other by selecting the commands to move and using the Add or Remove command buttons between the lists of commands.

A separator is a small space you can place between buttons. Each command can be used only once in the toolbar, but you can use more than one separator.

You can move the buttons around on the toolbar using the Move Up and Move Down command buttons to the right of the toolbar list.

Small, medium, or large icons or buttons can be selected at the bottom of the dialog box.

If you want to go back to the original toolbar configuration, press the Reset button.
