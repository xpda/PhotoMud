Image File Types

Photo Mud supports these image file types:

.bmp Windows Bitmap
.cur Windows Cursor
.emf Enhanced Metafile
.eps Encapsulated Postscript
.fpx FlashPix
.ico Windows Icon
.jpg JPEG
.pct Mac Pict
.png Portable Net Graphics
.psd Photoshop
.tga Targa
.tif Tagged Image File Format
.wmf Windows Metafile

You can select the file types to be displayed in Photo Mud Explorer with Select Image File Types under the View menu.
