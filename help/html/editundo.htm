Undo and Redo

You can undo editing operations in Photo Mud Editor using the Ctrl-Z shortcut or using the Undo command under the Edit menu. You can redo, or undo an undo, with Ctrl-Y or with the Redo commend under the Edit menu.

The multilevel Undo requires a fair amount of disk space. If this is a problem, you can disable multilevel Undo in the Photo Mud Options.
