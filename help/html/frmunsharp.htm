Sharpening an Image

In many cases, an image that is a little blurry can be substantially improved with the Photo Mud sharpening function. The Sharpen command uses an unsharp mask to do the sharpening. You can adjust three settings to customize the filtering.

  1. Amount -- The strength of the adjustment, or the amount each modified pixel is changed.

  2. Radius -- The radius of a pixel's neighborhood. This tells how far the program will look in determining whether a pixel should be adjusted.

  3. Threshold -- The threshold at which a pixel is modified; the minimum difference between a pixel and its blurred neighborhood.

The Sharpen command is under the Image menu of Photo Mud Editor.
