Photo Mud Explorer and Photo Mud Editor

Photo Mud has two main screens or windows, the Explorer and the Editor. Photo Mud Explorer is used to display and browse images, and to perform operations on multiple images such as printing and conversions. It is the window with a list of folders on the left and a list of files at the lower right.

Photo Mud Editor is the window used for editing and modifying a single image. It usually contains a single image. When you open a file in Photo Mud Explorer, it appears in Photo Mud Editor so you can edit, modify, and enhance the image.

You can move back and forth between Photo Mud Explorer and Photo Mud Editor using the F6 key.

When you start Photo Mud, it normally begins in Photo Mud Explorer window. However, if you start Photo Mud by double clicking on an image file in Windows Explorer, it will open the image in Photo Mud Editor window.

You can always tell whether you're in Photo Mud Editor or Explorer window by looking at the title bar at the top of the window.

It's easy to change the window configuration in Photo Mud Explorer. You can drag the bars between the current image and the fold and file lists to change the size of the folder and file lists. You can also specify whether to list the image files with thumbnail images, file details, or a list of file names.
