Edge Detection

Photo Mud provides several edge detection filters. As the name implies, these filters are used to detect edges and lines in an image. The resulting image is usually black with a few white places where the edges and lines are found.

In some cases, such as image analysis, this black image is the desired result. In other cases, you may want to add or subtract the detected edges back to the original image to make it look sharper. The Edge Detection dialog has command buttons for this, "Add to Original Image" and "Subtract from Original Image." The "Strength" slider tells how much of the edges to add into the original image.

These edge detection methods are available:

  • Gradient Directional

  • Laplacian Line Detection

  • Sobel Edge Detection

  • Prewitt Edge Detection

  • Shift-Difference Edge Detection

  • Line Segment Detection

The "Direction" option specified which direction for the filter to search for edges.

The Original and Modified images can be zoomed in or out using the command buttons between the images. You can drag the mouse across an image to move or pan the images. It may be useful to zoom in on the image for a better view of the changes being made.

Edge Detection is under the Image menu of Photo Mud Editor.
