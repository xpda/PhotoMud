Image Skew

The Image Skew function skews or slants the rectangular shape of an image into a parallelogram. You can slant the image up to 45 degrees either horizontally or vertically.

The Original and Modified images can be zoomed in or out using the command buttons between the images. You can drag the mouse across an image to move or pan the images. It may be useful to zoom in on the image to get a better idea of the changes being made. Each change skew angle resets the image view in the windows so you can see the full effect of the skew.

The Skew command is under the Image menu of Photo Mud Editor.
