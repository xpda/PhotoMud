Printing with Photo Mud

To Print with Photo Mud, use the Print command from the File menu. A number of standard printing options are available. You can print to any size and justification, print at landscape or portrait orientation, and specify values in inches or mm. A preview of the image on paper is automatically displayed.

You can print a group of images tagged in Photo Mud Explorer using Print Tagged Images from the File menu.

If you are printing more than one copy or a group of tagged images, you can select the option to print multiple images per page. You can then specify how many rows and columns of images to be printed on each page.
