Color Tolerance

The Color Tolerance is the range of pixel intensities to be included in the Fill and Select Similar Color commands. The larger the number, the wider the selection in the image.

A value of 255 would include the entire image, and a value of 0 would include only connected pixels of identical colors. 20 is a good value for general cases.

The Color Tolerance can be set with Similar Color Tolerance under the Edit menu or under the Options command under the Tools menu of Photo Mud Editor.
