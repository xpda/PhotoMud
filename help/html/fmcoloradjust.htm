Color Adjustment

The Color Adjustment option of the Convert Tagged Image function in Photo Mud Explorer can be used to set a color adjustment to the tagged files being converted. You can specify the brightness settings for the image, or for red, green, and blue independently. This makes it easy to modify a group of photos that are too dark, too red, etc.

You can click the arrow buttons to browse through the tagged images and preview the color changes.

The Gamma Correction checkbox causes the Gamma Correction method of brightness change. This produces a more realistic result in many cases. The Gamma Correction changes the color intensity with exponential function instead of the linear function of the ordinary Brightness function.
