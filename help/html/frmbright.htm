Brightness and Contrast

You can set the brightness and contrast by sliding the bars with the mouse or by entering the percent change, from -100 to 100. The Image brightness and contrast adjust all colors together. Alternatively, each color can be set independently.

The Gamma Correction checkbox causes the Gamma Correction method of brightness change. This produces a more realistic result in many cases. The Gamma Correction changes the color intensity with exponential function instead of the linear function of the ordinary Brightness function.

The Original and Modified images can be zoomed in or out using the command buttons between the images. You can drag the mouse across an image to move or pan the images. It may be useful to zoom in on the image to get a better idea of the changes being made.

The Brightness and Contrast command is in the Color menu of Photo Mud Editor.
