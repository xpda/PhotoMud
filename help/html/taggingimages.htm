Tagging Images

In the file list on Photo Mud Explorer screen there is a check box next to each file. You can click on these boxes to tag files for the batch processing commands Print Tagged Images, Email Tagged Images, and Convert or Resize Images.

A file is selected when it is highlighted. A file is tagged when its check box is checked. You can tag all selected files, select all tagged files, or clear all tags by choosing the appropriate command under the Edit menu.

To tag all the images in the current folder, press Ctrl-A to select all images then Ctrl-T to tag the selected images.

After you perform an operation on a set of tagged images, the image tags may be cleared. Image tags can also be cleared by pressing Esc. To restore a set of tags after they are cleared, use Set Previous Tags from the Edit Menu.

You can normally tag images in more than one folder, although images from only one folder can be listed at one time. You can restrict tags to only one folder in the Photo Mud Options.
