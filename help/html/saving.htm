Saving Images

Photo Mud can save an image in any of several different file formats. In Photo Mud Explorer, an image can be resized when it is saved. This is useful if you want to reduce the file size of an image for email or a web page.

To save an image, select the Save or Save As command under the File menu.

In Photo Mud Editor you can save a selected portion of an image using Save Selection under the File menu.
