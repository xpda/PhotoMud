Screen Capture

The Screen Capture grabs a section of the computer screen and converts it into a Photo Mud image. Then it can be saved, printed, or edited like any other image.

To capture a screen image, select Screen Capture from the File menu, and then click the Capture button. Then display the screen you want to capture. You can use Alt-Tab or the task bar to change to another Windows application.

When the desired screen is displayed, press F11. F11 is the "hot key" that begins the capture process. You'll see the cursor change to display the (X, Y) coordinates when the capture process starts.

To capture the screen image, drag a box around that portion of the screen you want to capture. When you release the mouse button, you will automatically return to Photo Mud Editor window with the new screen image.
