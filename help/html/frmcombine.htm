Combining Two Images

To combine two images, first open both images in Photo Mud. When more than one image is open, you can switch from one to another using Ctrl-F6. If more than two images are open, you will be able to select the second image to be used with the currently displayed image.

The Combine Images command is used to combine the two images into a single image. The image displayed on the active Photo Mud Editor window, Image 1, is replaced with the combined image.

If you check the "Scale to Fit" option, Image 2 (on top) will be scaled so it fits inside Image one. If you uncheck "Maintain Aspect Ratio," the height and width of Image 2 will be scaled to fit on top of Image 1.

If you want one of the images to be converted to grayscale for the combining, you can check that box below the image.

There are several combination methods that can be used to combine the images:

  • Add -- The corresponding pixels of the two images are added together. You may want to adjust the intensity of the images to produce the desired result. This is sometimes used with the edge detection filters to sharpen an image.

  • And -- This "And's" each pair of pixels. It is a logical bit operation than can produce some unusual colors for special effects.

  • Average -- Takes the average of corresponding pixels of the two original images.

  • Divide Image 1 -- Divides each pixel of Image 1 (on bottom) from the corresponding pixel in Image 2. This produces some unusual colors for special effects.

  • Divide Image 2 -- Divides each pixel of Image 2 (on top) from the corresponding pixel in Image 2. This produces some unusual colors for special effects.

  • Exclusive Or -- This "XOR's" each pair of pixels. It is a logical bit operation than can produce some unusual colors.

  • Maximum -- The resulting image uses the brightest pixel of each image.

  • Minimum -- The resulting image uses the darkest pixel of each image.

  • Multiply -- The corresponding pixels of the two images are multiplied together.

  • Or -- This "Or's" each pair of pixels. It is a logical bit operation than can produce some unusual colors for special effects.

  • Replace -- Replaces the pixels in image 1 with image 2. This just copies image 2 onto image 1.

  • Subtract Image 1 -- Subtracts each pixel of Image 1 (on bottom) from the corresponding pixel in Image 2. You may want to adjust the intensity of the images to produce the desired result. This is sometimes used with the edge detection filters to sharpen an image.

  • Subtract Image 2 -- Subtracts each pixel of Image 2 (on top) from the corresponding pixel in Image 1. You may want to adjust the intensity of the images to produce the desired result. This is sometimes used with the edge detection filters to sharpen an image.

The Red, Green, and Blue values of the pixels are combined independently.

Converting one image to grayscale is sometimes useful in combining images.

The results of edge detection can sometimes be combined with the original image to enhance the edges or sharpen the original image.

The Combine Images command is under the Tools menu of Photo Mud Editor.
