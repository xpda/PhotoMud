Photo Mud Options

You can specify the following options in the Options command under the Tools menu:

  • Image Folder -- The default folder for Photo Mud images.

  • Default File Format -- The default file format.

  • JPG Compression -- JPG Compression for saving images.

  • Email JPG Compression -- JPG Compression for email attachments used in the Email Image command.

  • Color Tolerance for Magic Wand Selection -- The Color Tolerance for the Fill and Select Similar Color commands.

  • Show Toolbar -- You can show or hide the toolbar.

  • Enable Multi-level Undo -- You can disable Photo Mud's undo and redo capability in order to save disk space. Each level of undo saves a copy of the image to disk temporarily.

  • Use Classic Help Style -- Photo Mud help is normally displayed using the newer html help interface. You can change to the classic Windows help if you prefer.

  • Allow Tags in Multiple Folders -- Photo Mud allows you to tag images from different folders in Photo Mud Explorer. If this option is not checked, then image tags will be cleared when you change from one folder to another.

  • Enable Mouse Wheel -- The mouse wheel is normally used to zoom in or out in Photo Mud Editor. This can be disabled if you prefer not to use it.

  • Begin Editing at 1:1 Zoom -- If this is checked, then an image loaded into Photo Mud Editor will be displayed at 1:1 zoom, where each pixel on the display shows one pixel of the image. If this is not checked, the image will be displayed at the largest "integral" zoom so that it fits on the screen.

  • Save Keywords in Comment Field -- This allows you to add keywords to the Exif Comment field instead of the Windows Keyword field in the File Comment command.

  • Show Tips at Startup -- If this is checked, the Tip of the Day will be displayed when Photo Mud starts.

  • Foreground Color -- The color used with text and drawing commands.

  • Background Color -- The background color for drawing commands and new images. This determines the background color of new images -- it does not affect the color of an image already loaded.

  • Border Color -- The border color for image borders.

  • Slide Show Options
    • Frames per Minute -- The number of images to be shown per minute. For example, 30 frames per minute will display 1 image every two seconds in the slideshow.

    • Random Order -- Check this option to show the tagged images in random order. Otherwise, the images are displayed in the order in which they were tagged.

    • Show File Name -- Display the file name along with the images in the slide show.

    • Show Controls -- Display the slideshow control box.

Reset to Defaults returns all Photo Mud options to the program defaults.
