Making a Web Page

Photo Mud can automatically create a web page containing tagged images. The web page will contain a table of thumbnail images and optional captions. Any thumbnail image on the web page can be clicked on to view the full-size image. Here's how to do it:

  • In Photo Mud Explorer, tag the images you want to be included in the web page.

  • Select Make Thumbnail Web Page from the Tools menu.

  • Enter any captions for the images.

  • Enter any customizing information in the Web Page dialog box.

  • Click the "Save Web Page" button.

It's a good idea to save the web page in a separate folder. When you save a web page, three things are normally saved:

  • The HTML file. This is the web page itself. It contains references to the images and the thumbnail images.

  • The thumbnail images. These are the small images that appear in the web page.

  • The image files. These are the full-sized images, or they can be resized copies the originals.

After you have saved a web page, you can view it in your web browser by clicking on the "View Web Page" button. If you need to make changes, you may want to check the option "Save HTML file only, no images," if the images are correct.

Note: You can make changes to the web page and save it again as long as you stay in the Web Page dialog. After you click "Close" and go back to Photo Mud Explorer screen you will no longer be able to made modifications to the web page, although you can overwrite it.

The web page options you select will be retained for the next web page you make. If you want to revert to the program defaults, click on the button labeled "Use Default Settings."

Thumbnail Images

The thumbnail images used in the web page are small copies of the original images. They are copied to the same folder as the HTML file, with "small_" added onto the original image file name. When you make a web page, the thumbnail images appear in the page. Then when you browse the web page, you can click on a thumbnail image to see the large version of the image.

Images and Captions

In the upper right corner of the Web Page screen there is a set of boxes that represent the images on the web page. You can click on any of the boxes to see the image and optionally add an image caption to be shown on the web page. You can also select an image in the image list below the image arrangement boxes.

If you would like to reposition an image, use the up and down arrow buttons to the right of the image list. This will move the current image up or down one position.

Web Page Options

Number of Columns is the number of columns in the table for the thumbnail images. Depending on the thumbnail image size and the resolution of the web browser's screen, this will determine whether all the images will fit on the screen without scrolling horizontally. Normally in a web page it is better to scroll vertically than horizontally. A quick way to tell whether the thumbnails fit across the screen is to View the Web Page after you make it, and then come back and adjust this option, the thumbnail size, or the spacing options if necessary.

Max Horizontal Size and Max Vertical Size are the maximum possible horizontal and vertical size, in pixels, of the thumbnails. The thumbnail images will be fit within these size limits without "stretching" or adjusting the aspect ratio of the images.

Cell Border Width is the thickness of the border between the images. The HTML term for this option is "tableborder."

Space Around Cell Contents is the space between the images and the cell borders. The HTML term for this option is "cellpadding."

Space Between Cells is the space between the cells. The HTML term for this option is "cellspacing."

It's easy to experiment with these three options using the View Web Page button. You can use the "Save HTML File Only" option after you have created the thumbnails the first time. If you set the Cell Border Width to zero the borders between the images will be invisible.

Title Font Size and Caption Font Size are the font sizes for the text in the web page. The Font Name can be specified, or it can be left blank to use the default browser font.

The Background Color and Text Color can be set by clicking on the color box.

The Web Page Title is the title that is displayed at the top of the web page.

You can click on the buttons inside the "Add to All Captions" box to add the Image Comments, Dates, and Names to the captions in the web page. Image Comments are internal comments in the image file. You can edit these using the Image Comments command. The Image Date is the date inside the file that tells when a photo was taken or an image was created. It may be different from the file date. Image Comments and Image Dates are not available in all image files.

If "Save HTML File Only" is checked, then the images will not be saved. This is useful for modifying the web page parameters after the thumbnail images have been created. However, the web page cannot be modified after exiting the "Make Web Page" screen.

Save Web Page Images

The small thumbnail images that Photo Mud creates are always saved with the web page. Then Photo Mud will ask if you want to save the web page images. If so, the original images will be copied to the folder with the web page itself. You can optionally resize the images when they are copied.

The web page images should normally be saved unless they are already in the folder where you save the web page.
