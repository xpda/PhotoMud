Image Information

The Image Information function displays detailed information about an image. Most digital cameras include EXIF information in the images that includes the camera model, date and time the picture was taken, shutter speed, etc.

The following information is shown for most images:

  • File Type -- The file type includes more information than the file extension. For example, there are several types of .jpg files.

  • Comment -- The user comment is displayed if there is one. You can add a user comment to any .jpg, .tif, or .png image.

  • Bits per Pixel -- The number of bits that make up each pixel in the image.

  • Compression -- The type of compression used when the image is saved on disk.

  • Image Height -- Height of the image in pixels.

  • Image Width -- Width of the image in pixels.

  • Image Size on Disk -- The amount of space the image occupies on disk.

  • Image Size in Memory -- The amount of RAM required to hold the image while it is being used in a program.

  • Resolution -- The X and Y resolutions are the number of pixels per inch horizontally and vertically. These are somewhat arbitrary values since the image can be displayed and printed at any size.

The following EXIF Information is commonly included for images from modern digital cameras, although different cameras provide different information.

  • Date and Time -- Date and time the photo was taken.

  • Make -- Camera manufacturer.

  • Model -- Camera model.

  • Software -- Internal camera software version.

  • Shutter Speed -- Shutter speed in 1/1000's of a second.

  • Aperture

  • Exposure Bias

  • Max Aperture

  • Subject Distance (meters).

  • Metering Mode.

  • Flash -- Whether the flash was fired.

  • Focal Length

  • Exposure Time -- Exposure time, in seconds.

  • F-Stop

  • ISO Speed Ratings

  • Maker Note -- The Maker Note area of the image may include additional camera parameters and settings. Maker notes are proprietary manufacturer information. The data is not well documented, and the information may vary among camera models. In other words, information presented in the maker notes may be unreliable.

On the Image Information screen you can browse from one image to the next in the current folder by clicking on the arrow buttons beneath the file name.

If a digital photo is saved by a program that does not support EXIF information, that information may be lost when it is saved. The EXIF information is available in .jpg and .tif files.

You can click the Copy button to copy this information to the clipboard so you can paste it into other applications, and you can click the Save button to save the information into a text file.

The Image Information command is under the Tools menu.
